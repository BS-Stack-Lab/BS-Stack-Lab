### 🚀 AI-Assisted Portfolio
> **"기획부터 암호화 저장까지, AI와 함께 빌드한 서비스"**

#### 1️⃣ Personal Data Vault (암호화 서비스)
- **Concept:** 개인 정보를 입력받아 해시 함수 및 암호화 로직을 거쳐 DB에 안전하게 저장하는 보안 서비스.
- **Key Logic:** Java 기반 데이터 처리 + AI를 활용한 보안 로직 검증.
- **Result:** 디자인 전공자로서 기획한 **보안 UX**와 백엔드의 **데이터 보호** 가치를 결합.

#### 2️⃣ Logic & Fun: Text-based Game & Calculator
- **Concept:** 순수 Java 로직으로 구현한 인터랙티브 텍스트 게임 및 계산기.
- **Focus:** AI를 도구로 사용하여 복잡한 알고리즘을 효율적으로 구현하고, 코드의 가독성을 디자인적 관점에서 정리.
